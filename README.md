# cost-of-me

cost-of-me は、自分が生まれてから社会に出るまでに、  
親がどれくらいのお金を使ってくれたのかを概算する日本向けWebアプリです。

簡単な質問に答えるだけで、教育費・生活費・習い事などをもとに  
育てるためにかかったコストを可視化します。

このツールは正確な家計簿を再現するものではなく、  
親への感謝や親孝行を考えるきっかけを作ることを目的としています。

---

## Concept

親が自分を育てるためにどれくらいのお金を使ってくれたのか、  
正確に知る機会はあまりありません。

cost-of-me は、簡単な質問に答えるだけで、  
その金額を概算し可視化するツールです。

このツールを通して、  
親への感謝や親孝行について考えるきっかけになればと思っています。

---

## How it works

以下のような簡単な質問に答えるだけで計算できます。

- 中学は公立？私立？
- 高校は公立？私立？
- 大学は？
- 塾に通っていた？
- 食費は1食いくらくらい？
- 一人暮らしはした？
- 習い事はしていた？
- 病院によく通っていた？

これらの回答をもとに、  
一般的な統計や相場を組み合わせて育成コストを概算します。

---

## Result Example

結果では以下のような情報を表示します。

- 総額
- 内訳（教育費 / 生活費 / 習い事 / 医療費など）
- 身近なものとの比較

例：

あなたの親は  
**約 2,340 万円** を  
あなたのために使っています。

これは

- 🚗 新車 約6台分
- 🏠 東京の家賃 約12年分
- 💼 日本の平均年収 約5年分

に相当します。

## Share

計算結果はSNSでシェアできます。

例：

私の cost-of-me は  
**2,340万円** でした。

新車 約6台分らしいです。  
親に感謝。

#costofme

---

## Notes

- 表示される金額は概算です  
- 家庭ごとの事情を完全に再現するものではありません  
- 正確な支出記録ではなく、気づきのきっかけを作るツールです
